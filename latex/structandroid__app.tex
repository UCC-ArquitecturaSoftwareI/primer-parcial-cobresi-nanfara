\doxysubsection{Descripción detallada}
This is the interface for the standard glue code of a threaded application. In this model, the application\textquotesingle{}s code is running in its own thread separate from the main thread of the process. It is not required that this thread be associated with the Java VM, although it will need to be in order to make J\+NI calls any Java objects.
