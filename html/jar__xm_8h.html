Tankey Game  1.0
Tankey es un juego sobre un gorila. Este gorila trepa por un árbol infinito, pero a medida que sube se enfrenta a un problema: debe esquivar las ramas que se avecinan sobre él. Moviéndose de un lado al otro... ¿Podrá lograrlo?

◆ jar_xm_get_remaining_samples()

uint64_t jar_xm_get_remaining_samples ( jar_xm_context_t *  ctx)

Get the number of remaining samples. Divide by 2 to get the number of individual LR data samples.

Nota
This is the remaining number of samples before the loop starts module again, or halts if on last pass.
This function is very slow and should only be run once, if at all.
